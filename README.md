## upd是个什么项目？

提供文本浏览、超文本浏览、文件上传以及代码执行的功能。

文件上传和代码执行是需要持有私钥的。

## upd的加密原理是什么？

首先先向web服务器请求AES密钥，服务器会将密钥通过RSA公钥加密。也就是说，你需要事先下载私钥才能解密。

然后，下一个文件上传、代码执行视图的荷载将会使用这个AES密钥进行加密传输，随后服务器会刷新AES密钥。

## 文件浏览功能如何使用？

访问程序监听的web端口，该端口在config.ini中进行配置，通常是9200。你将会看到一个目录，它被映射到项目目录的html文件夹下面。

你可以点击文件进行查看，可以点击进入文件夹，还可以浏览网页。

## html目录下默认都有哪些文件？

html目录即是文件浏览的主目录，它下面我放了一些常用的页面，比如文件上传、代码执行、代码编辑，常用的脚本以及**mybot**项目使用的动作脚本。
